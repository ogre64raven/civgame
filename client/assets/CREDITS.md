## 동물·부족 모델 추가 방법 (선택)

아래 파일명으로 GLB를 넣으면 서버 재시작 없이 자동 인식됩니다 (없으면 기본 도형 사용):

```
client/assets/neutral/wolf.glb
client/assets/neutral/bear.glb
client/assets/neutral/tiger.glb
client/assets/neutral/lion.glb
client/assets/neutral/tribe.glb
client/assets/units/character.glb  (인간 유닛 교체)
```

추천 CC0 팩 (수동 다운로드):
- Quaternius — Animated Animals / Ultimate Animals: https://quaternius.com (CC0)
- KayKit — Adventurers (인간 캐릭터): https://kaylousberg.itch.io/kaykit-adventurers (CC0)
- Kenney — 모든 팩: https://kenney.nl/assets (CC0)

rigged(스켈레탈) 모델도 지원하며 'idle'/'walk' 클립이 있으면 자동 재생됩니다.
